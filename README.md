# Personal Portfolio Website

A sleek, modern, and fully responsive personal portfolio website designed to showcase my skills, services, projects, and testimonials. Built with a clean UI/UX layout featuring a custom color palette, smooth navigation, and premium interactive hover elements.

## ✨ Features
- **Modern UI/UX:** Built with a clean geometric layout using custom polygons and a professional color scheme.
- **Smooth Navigation:** Includes a sticky header with a modern frosted-glass blur (`backdrop-filter`) effect and smooth section scrolling.
- **Dynamic Skill Bars:** Visually represents expertise across both Web Development and Core Programming stacks.
- **Interactive Components:** Interactive service cards and dynamic hover transformations across the portfolio grids.
- **Contact Form:** Integrated clean and styled template layout for user messages and collaborations.

## 🛠️ Tech Stack & Tools
- **Frontend:** HTML5, CSS3 (Custom Grid/Flexbox layouts), JavaScript (ES6)
- **Icons & Fonts:** FontAwesome Icons, Google Fonts (Poppins)
- **Design:** Prototyped and measured using Figma
